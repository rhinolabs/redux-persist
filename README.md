# Redux Persist
Subscribe the Redux Store and replicate to `sessionStorage`.

**Features**
- Persist the redux store in `sessionStorage`

## Installing
```shell
# install via npm
$ npm install @rhinolabs/redux-persist
#or yarn
$ yarn add @rhinolabs/redux-persist
```
